INF02 Teoria Linux - Polecenia

Podstawowe polecenia Linux - cz.1

man - wyświetla pomoc do wybranego polecenia
Przykład: man ls
pwd - wyświetla lokalizację w której jesteśmy
cd .. - wychodzimy z katalogu
cd administrator - wchodzimy na katalog administratora
cd / - wychodzimy ze wszystkiego
cd ~ - wracamy do podstaw np /home
ls - wyświetla katalogi i foldery oraz pliki
ls -la - lista katalogów
mkdir - tworzenie katalogu
rmdir (nazwa) - usuwa katalog (musi być pusty)
touch - tworzy plik
rm - usuwa plik
rm -r (nazwa) - usuwa katalog razem z zawartością
cat (nazwa pliku) - wyświetla zawartość i komendy które były używane w tym pliku
cp (nazwa pliku) (dokąd) - kopiowanie pliku
mv - pozwala zmienić nazwę jak i przenieść plik
sudo cfdisk - partycje dysków itd
fdisk - to samo co cfdisk lecz więcej opcji
sudo fdisk /dev/sda - zarządzanie dyskiem
sudo mkfs.ext4 (nazwa dysku) - zmienia system plików dysku
sudo chown (user):(grupa) (katalog) - zmiana właściciela i grupy katalogu

Konfiguracja Linux Server - cz.2

lsb_release -d - sprawdzenie nazwy dystrybucji
echo "nazwa dystrybucji:" > SO - utworzy plik SO
lsb_release -d >> SO - skopiuje zawartość pliku do pliku SO
UWAGA!
Jeśli chcemy sprawdzić czy na pewno wykonaliśmy powyższe czynności dobrze wpisujemy:
nano SO
W ten sposób wyświetli nam się zawartość pliku SO
uname --help - wyświetli przystawki które pomogą nam sprawdzić wersję jądra systemu itd.
uname -rv - wyświetla wersję jądra i datę jego wydania.
echo "wersja jądra:" >> SO - dodaje nagłówek do pliku SO
UWAGA!
NIE WPISUJEMY POJEDYNCZEGO ZNAKU WIĘKSZOŚCI, W PRZECIWNYM RAZIE NADPISZEMY NASZ PLIK SO
uname -m - architektura systemu (w Linuxie jest podpisana jako machine)

Procesor

lscpu - informacje o procesorze
lscpu | grep "Model name" - wyświetli model procesora
lscpu | grep "Model name" > CPU - utworzy plik CPU
lscpu | grep "GHz" >> CPU - informacja o taktowaniu procesora
UWAGA!
JEŚLI POTRZEBUJEMY INNYCH WARTOŚCI JAK PAMIĘĆ CACHE L1 LUB L2 WPISUJEMY PO KOMENDZIE GREP W ZNACZNIKACH L1 BĄDŹ L2

Pamięć RAM

sudo lshw -c memory - wyświetla informacje o kościach w bankach pamięci
sudo lshw -c memory | grep GiB > RAM - zapisuje pojemność RAM do pliku

Płyta główna

sudo dmidecode -t baseboard - wyświetla info o płycie głównej
sudo dmidecode -t baseboard | grep Manufacturer > MAINBOARD - producent płyty głównej
sudo dmidecode -t baseboard | grep Serial >> MAINBOARD - numer seryjny płyty
lsblk - wyświetla dostępne dyski
sudo mount /dev/sdb1 /mnt/pen - montowanie dysku (np. pendrive'a)
sudo cp SO /mnt/pen - kopiowanie plików na zamontowany dysk

Konfiguracja dysku

sudo blkid >> /etc/fstab - dodanie UUID dysku do fstab
Format wpisu w fstab:
UUID (NAZWA_KATALOGU) (FORMAT) (PARAMETR np. nofail)

Konfiguracja Apache

nano /etc/apache2/ports.conf - edycja portów (zmiana w linijce listen)
nano /etc/apache2/sites-enabled/000-default.conf - konfiguracja witryny
Zmiana katalogu źródłowego witryny:
DocumentRoot (nazwa_katalogu)
sudo systemctl restart apache2 - restart serwera WWW

Polityka haseł

chage --help - wyświetla opcje zarządzania hasłami
sudo passwd (nazwa_uzytkownika) - zmiana hasła dla użytkownika
